# Spec: Egress Domain Allowlisting

## Overview

Enforce egress domain restrictions from `conga-policy.yaml` across all three providers, closing the #1 security gap. Uses the policy schema from Spec 1 (`cli/pkg/policy/`) for data model and validation.

**Unified enforcement mechanism:** Per-agent nginx SNI filtering proxy on all providers. One proxy container per agent, on the agent's own Docker network. Same code path everywhere — only the transport differs (local Docker CLI, SSH, bootstrap script).

**Why per-agent, not shared:** A shared proxy serving all agents requires a union allowlist, which leaks one agent's allowed domains to all others. Per-agent proxies give true egress isolation that's consistent from local through production. The cost is ~5MB memory per proxy container — negligible.

**Source**: `portable-policy.md` sections 1.2, 2.2; `product-knowledge/standards/security.md` Egress Policy + Enforcement Escalation

## Architecture

Each agent gets its own nginx proxy with its own allowlist derived from `MergeForAgent()`. Proxies live on the agent's Docker network — no shared egress network needed.

```go
package policy

import (
	"fmt"
	"path/filepath"
	"strings"
)

// LoadEgressPolicy loads the policy file, merges for the given agent, and returns
// the effective egress policy. Returns nil, nil if no policy file or no egress section.
func LoadEgressPolicy(policyDir string, agentName string) (*EgressPolicy, error) {
	policyPath := filepath.Join(policyDir, "conga-policy.yaml")
	pf, err := Load(policyPath)
	if err != nil {
		return nil, fmt.Errorf("loading policy: %w", err)
	}
	if pf == nil {
		return nil, nil
	}
	if err := pf.Validate(); err != nil {
		return nil, fmt.Errorf("invalid policy: %w", err)
	}

	effective := pf.MergeForAgent(agentName)
	return effective.Egress, nil
}

// EffectiveAllowedDomains returns the list of domains an agent can reach,
// excluding any that appear in blocked_domains. Blocked takes precedence.
func EffectiveAllowedDomains(e *EgressPolicy) []string {
	if e == nil || len(e.AllowedDomains) == 0 {
		return nil
	}
	blocked := make(map[string]bool, len(e.BlockedDomains))
	for _, d := range e.BlockedDomains {
		blocked[strings.ToLower(d)] = true
	}
	var result []string
	for _, d := range e.AllowedDomains {
		if !blocked[strings.ToLower(d)] {
			result = append(result, d)
		}
	}
	return result
}

// EgressProxyName returns the proxy container name for an agent.
func EgressProxyName(agentName string) string {
	return "conga-egress-" + agentName
}

// GenerateNginxAllowlist generates an nginx map block for SNI-based domain filtering.
// Wildcards (*.example.com) become regex patterns. Unmatched hostnames get empty
// $upstream which causes nginx to close the connection (502 to client).
func GenerateNginxAllowlist(domains []string) string {
	var b strings.Builder
	b.WriteString("# Generated by conga — do not edit manually\n")
	b.WriteString("map $ssl_preread_server_name $upstream {\n")
	b.WriteString("    default \"\";\n") // empty = reject

	for _, d := range domains {
		d = strings.ToLower(d)
		if strings.HasPrefix(d, "*.") {
			base := strings.TrimPrefix(d, "*.")
			escaped := strings.ReplaceAll(base, ".", "\\.")
			b.WriteString(fmt.Sprintf("    \"~^(.+\\.)?%s$\" $ssl_preread_server_name:443;\n", escaped))
		} else {
			b.WriteString(fmt.Sprintf("    %s %s:443;\n", d, d))
		}
	}

	b.WriteString("}\n")
	return b.String()
}

// GenerateNginxConf generates the full nginx.conf for an egress proxy with domain filtering.
func GenerateNginxConf(domains []string) string {
	var b strings.Builder

	b.WriteString("# Conga Line Egress Proxy — per-agent domain filtering\n")
	b.WriteString("# Generated by conga — do not edit manually\n\n")
	b.WriteString("events {\n    worker_connections 256;\n}\n\n")
	b.WriteString("stream {\n")

	// Log format for denied connections (empty upstream = denied)
	b.WriteString("    log_format egress '$remote_addr [$time_local] '\n")
	b.WriteString("                      '$ssl_preread_server_name -> $upstream '\n")
	b.WriteString("                      '$status $bytes_sent $bytes_received';\n")
	b.WriteString("    access_log /dev/stdout egress;\n\n")

	if len(domains) > 0 {
		// Allowlist mode: only listed domains pass
		b.WriteString("    map $ssl_preread_server_name $upstream {\n")
		b.WriteString("        default \"\";\n")
		for _, d := range domains {
			d = strings.ToLower(d)
			if strings.HasPrefix(d, "*.") {
				base := strings.TrimPrefix(d, "*.")
				escaped := strings.ReplaceAll(base, ".", "\\.")
				b.WriteString(fmt.Sprintf("        \"~^(.+\\.)?%s$\" $ssl_preread_server_name:443;\n", escaped))
			} else {
				b.WriteString(fmt.Sprintf("        %s %s:443;\n", d, d))
			}
		}
		b.WriteString("    }\n\n")
	} else {
		// Passthrough mode: all domains pass
		b.WriteString("    map $ssl_preread_server_name $upstream {\n")
		b.WriteString("        default $ssl_preread_server_name:443;\n")
		b.WriteString("    }\n\n")
	}

	// HTTPS proxy
	b.WriteString("    server {\n")
	b.WriteString("        listen 3128;\n")
	b.WriteString("        proxy_pass $upstream;\n")
	b.WriteString("        ssl_preread on;\n")
	b.WriteString("        resolver 8.8.8.8 valid=300s;\n")
	b.WriteString("        resolver_timeout 5s;\n")
	b.WriteString("    }\n\n")

	// DNS forwarding
	b.WriteString("    server {\n")
	b.WriteString("        listen 53 udp;\n")
	b.WriteString("        proxy_pass 8.8.8.8:53;\n")
	b.WriteString("        proxy_timeout 5s;\n")
	b.WriteString("    }\n\n")

	b.WriteString("    server {\n")
	b.WriteString("        listen 53;\n")
	b.WriteString("        proxy_pass 8.8.8.8:53;\n")
	b.WriteString("        proxy_timeout 5s;\n")
	b.WriteString("    }\n")

	b.WriteString("}\n")
	return b.String()
}
```

### 2. All Providers: Per-Agent Egress Proxy

The enforcement pattern is identical across providers. Each provider implements two functions using its own transport:

#### `startAgentEgressProxy(ctx, agentName, domains)`

1. Generate `nginx.conf` from `GenerateNginxConf(domains)`
2. Write config to provider-appropriate path:
   - Local: `~/.conga/config/egress-{agentName}.conf`
   - Remote: `/opt/conga/config/egress-{agentName}.conf` (via SFTP)
   - AWS: `/opt/conga/config/egress-{agentName}.conf` (via bootstrap)
3. Start nginx container:
   ```
   docker run -d \
     --name conga-egress-{agentName} \
     --network conga-{agentName} \
     --cap-drop ALL \
     --security-opt no-new-privileges \
     --memory 64m \
     --read-only \
     -v {confPath}:/etc/nginx/nginx.conf:ro \
     nginx:alpine
   ```
4. No port publishing — proxy is only reachable within the agent's Docker network

#### `stopAgentEgressProxy(ctx, agentName)`

1. `docker rm -f conga-egress-{agentName}`

#### Wiring agent containers

When egress is enforced, add to `runAgentContainer` args:

```
-e HTTPS_PROXY=http://conga-egress-{agentName}:3128
-e HTTP_PROXY=http://conga-egress-{agentName}:3128
-e NO_PROXY=localhost,127.0.0.1
--dns {proxy-container-IP}
```

The proxy container IP is resolved at runtime via `docker inspect --format '{{range .NetworkSettings.Networks}}{{.IPAddress}}{{end}}' conga-egress-{agentName}`.

#### New method: `startAgentEgressProxy` / `stopAgentEgressProxy`

```go
func (p *LocalProvider) startAgentEgressProxy(ctx context.Context, agentName string, domains []string) error {
    proxyName := policy.EgressProxyName(agentName)
    netName := networkName(agentName)

    // Stop existing proxy if any
    if containerExists(ctx, proxyName) {
        removeContainer(ctx, proxyName)
    }

    // Generate nginx config
    conf := policy.GenerateNginxConf(domains)
    confPath := filepath.Join(p.configDir(), fmt.Sprintf("egress-%s.conf", agentName))
    if err := os.WriteFile(confPath, []byte(conf), 0644); err != nil {
        return fmt.Errorf("writing egress config: %w", err)
    }

    // Ensure network exists
    if !networkExists(ctx, netName) {
        if err := createNetwork(ctx, netName); err != nil {
            return fmt.Errorf("creating network: %w", err)
        }
    }

    // Start proxy on agent's network
    _, err := dockerRun(ctx, "run", "-d",
        "--name", proxyName,
        "--network", netName,
        "--cap-drop", "ALL",
        "--security-opt", "no-new-privileges",
        "--memory", "64m",
        "--read-only",
        "-v", fmt.Sprintf("%s:/etc/nginx/nginx.conf:ro", confPath),
        "nginx:alpine",
    )
    if err != nil {
        return fmt.Errorf("starting egress proxy: %w", err)
    }

    fmt.Printf("  Egress proxy started for %s (%d domains allowed)\n", agentName, len(domains))
    return nil
}

func (p *LocalProvider) stopAgentEgressProxy(ctx context.Context, agentName string) {
    proxyName := policy.EgressProxyName(agentName)
    if containerExists(ctx, proxyName) {
        removeContainer(ctx, proxyName)
    }
}
```

func (p *RemoteProvider) startAgentEgressProxy(ctx context.Context, agentName string, domains []string) error {
    proxyName := policy.EgressProxyName(agentName)
    netName := "conga-" + agentName

    // Stop existing
    p.ssh.Run(ctx, fmt.Sprintf("docker rm -f %s 2>/dev/null || true", shellQuote(proxyName)))

    // Generate and upload config
    conf := policy.GenerateNginxConf(domains)
    confPath := fmt.Sprintf("/opt/conga/config/egress-%s.conf", agentName)
    if err := p.ssh.WriteFile(ctx, confPath, []byte(conf), 0644); err != nil {
        return fmt.Errorf("uploading egress config: %w", err)
    }

    // Start proxy
    cmd := fmt.Sprintf("docker run -d --name %s --network %s "+
        "--cap-drop ALL --security-opt no-new-privileges --memory 64m --read-only "+
        "-v %s:/etc/nginx/nginx.conf:ro nginx:alpine",
        shellQuote(proxyName), shellQuote(netName), shellQuote(confPath))

    if _, err := p.ssh.Run(ctx, cmd); err != nil {
        return fmt.Errorf("starting egress proxy: %w", err)
    }
    return nil
}

func (p *RemoteProvider) stopAgentEgressProxy(ctx context.Context, agentName string) {
    proxyName := policy.EgressProxyName(agentName)
    p.ssh.Run(ctx, fmt.Sprintf("docker rm -f %s 2>/dev/null || true", shellQuote(proxyName)))
}

### 6. Shared Egress Proxy Image

The existing `deploy/egress-proxy/` Dockerfile and image build are no longer needed as the per-agent enforcement mechanism. Per-agent proxies use `nginx:alpine` directly with a generated `nginx.conf` mounted in.

**Remove** the shared `ensureEgressProxy()` calls from both local and remote providers. Replace with per-agent proxy management. The old `conga-egress-proxy` container, `conga-egress` network, and `deploy/egress-proxy/` directory become unused.

**Keep `deploy/egress-proxy/`** as reference/documentation but don't build or deploy it. The `nginx:alpine` image is pulled directly.

## Edge Cases

| Scenario | Behavior |
|---|---|
| No policy file | All providers: behavior unchanged. No proxy started. No warnings. |
| Policy with no egress section | No enforcement. No warnings. |
| Egress with empty `allowed_domains` | No enforcement — empty allowlist means "no restrictions". Proxy runs in passthrough mode. |
| `mode: validate` with domains (local) | Warning printed to stderr. Agent starts normally. No proxy. |
| `mode: enforce` with domains (local) | Per-agent proxy started with allowlist. Agent wired through proxy. |
| `mode` unset (remote/AWS) | Enforce when domains are defined (remote/AWS always enforce). |
| `blocked_domains` overlaps `allowed_domains` | `EffectiveAllowedDomains()` removes blocked entries. Domain is blocked. |
| Wildcard `*.slack.com` | nginx regex `~^(.+\.)?slack\.com$` matches subdomains, not `slack.com` itself. |
| nginx:alpine image not available | `docker run` fails. Agent provision fails with clear error. Operator must `docker pull nginx:alpine`. |
| Per-agent override with different domains | Each agent's proxy gets its own allowlist. True per-agent isolation. |
| Agent removed | `stopAgentEgressProxy` removes the proxy container alongside agent cleanup. |
| RefreshAgent with changed policy | Old proxy stopped, new one started with updated allowlist. |
| Proxy container crashes | Agent traffic fails (HTTPS_PROXY unreachable). Operator runs `conga refresh`. |
| Agent provisioned before policy exists | No proxy. After creating policy, `conga refresh` starts proxy. |

## What This Does NOT Do

- No changes to the `conga-policy.yaml` schema (Spec 1 consumed as-is)
- No changes to the Provider interface
- No TLS termination (proxy inspects SNI hostname only)
- No Squid proxy (nginx everywhere for consistency; Squid is a future upgrade if logging needs grow)
- No iptables (per-agent nginx proxy is more consistent across providers)
- No policy file auto-creation (operator creates manually from .example)
- No shared proxy (per-agent for true isolation)
