# Security Policy

## Reporting a Vulnerability

Please use [GitHub's private vulnerability reporting](https://github.com/cruxdigital-llc/openclaw/security/advisories/new) to report security issues. Do not open public issues for vulnerabilities.

## Scope

We are interested in reports related to:

- Infrastructure misconfiguration
- Credential or secret exposure
- Container isolation failures
- IAM policy gaps

## Security Model

See [product-knowledge/standards/security.md](product-knowledge/standards/security.md) for details on the security architecture and threat model.
